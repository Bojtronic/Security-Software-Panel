El sistema debe realizar lo mismo que el sistema actual:

Al detectar que una persona se acerca a las puertas (sensores de proximidad) las puertas se liberan para poder abrirse 
Si se detecta la persona pero esta no abre la puerta, despues de 5 segundo las puertas se vuelven a bloquear
Si la persona abre la puerta, cuando la puerta regresa a su posición (cerrada) esta se vuelve a bloquear
Lo anterior se cumple para horario de trabajo (9am a 5pm) de lo contrario solo permite salir pero no permite entrar
Con el control remoto el cual tiene varios botones se pueden enviar comando para liberar o bloquear puertas, asi como para enviar señales de alerta a seguridad



Mejoras requeridas:

El hardware se debe integrar; los modulos de relés y fuente de poder en una sola tarjeta
Si se necesita hacer algun ajuste al horario de funcionamiento se busca poder hacerlo sin necesidad de abrir el panel, preferiblemente desde una laptop

Se requiere que con el control remoto se pueda bloquear o desbloquear completamente independientemente de presencia de personas

Buscar una alternativa a los controles remotos y receptores actuales
